502

Bad Gateway

The server received an invalid response from an upstream server while trying to fulfill your request.

What's happening?

  • The web server is having trouble communicating with the application server
  • PHP-FPM or another backend service may be down or overloaded
  • This is typically a temporary issue

What can you do?

  • • Wait a moment and refresh the page
  • • The service should resume shortly
  • • If the problem persists, contact the administrator
Homepage

If you're the site administrator, check your backend services (PHP-FPM, application servers).
